HOW TO PLAY

DroneStrike needs a projector and a game admin to run the game for the thrower.

  1. Game admin clicks the projector icon to open the target window, then projects that window over the physical targets.
  2. On the target window, click the M3 logo for projector options (target area, fullscreen, etc.).
  3. Game admin clicks LAUNCH or presses Spacebar to start a drone.
  4. When the thrower throws, the game admin presses Spacebar as the knife impacts the target — this freezes the drone.
  5. If the throw hit the drone, the game admin clicks the drone on the game control window (not the target / projector window) to score the hit.
